# S.A.F.E (Sistema autônomo de ferramentas estudantis)

Este projeto foi o objetivo do projeto integrador do Curso Técnico em informática do Instituto Federal de Santa Catarina Campus Gaspar.

O objetivo do trabalho foi entregar uma solução para os estudantes que pudesse facilitar seu dia a dia, fornecendo as ferramentas que eles mais utilizam em um único ambiente.

O projeto entrega ferramentas como:

- Bloco de notas;
- Calculadora avançada com fórmulas matemáticas mais utilizadas;
- Calendário de eventos com notificação;
- Organizador de disciplinas.

As principais técnologias utilizadas foram:

- SQLite;
- Java;
- Material design.
